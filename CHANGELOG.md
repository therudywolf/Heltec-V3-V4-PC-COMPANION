# Changelog

## v2.0 (Ultimate build)

- **11 экранов:** Main, Cores, CPU Details, GPU Details, Memory, Storage, Media, Cooling, Weather, Top CPU, Equalizer
- **Железо:** CPU/GPU температура и нагрузка, hotspot GPU, мощность CPU/GPU, вентиляторы (помпа, радиатор, корпус, GPU), частоты ядер и fallback CPU MHz, RAM/VRAM, диски (температура и использование)
- **Медиа:** обложка трека (48×48), исполнитель и название; обложка по чанкам на сервере; шрифт с поддержкой кириллицы
- **Погода:** Open-Meteo, текущая/день/неделя, иконки и описания
- **Процессы:** топ-3 по CPU, топ-2 по RAM (без MemCompression)
- **Сервер (monitor.py):** LHM, погода, медиа (winsdk), TCP с fallback, логирование ошибок, иконка в трее (pystray), автозапуск
- **UI:** единый стиль карточек, скруглённые прогресс-бары, контраст и карусель в меню, эквалайзер (Bars/Wave/Circle)
- **Сплэш:** одна фаза (текст + прогресс-бар)

Мини-игра Wolf & Moon в v2.0 не входит (убрана для упрощения и экономии памяти).
