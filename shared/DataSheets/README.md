# Даташит платы Heltec WiFi LoRa 32 V4

Для распиновки, схем питания и интерфейсов используется даташит **WiFi_LoRa_32_V4.2.0.pdf**.

- **Локально:** если файл `WiFi_LoRa_32_V4.2.0.pdf` лежит в этой папке — откройте его.
- **Скачать:** официальный сайт Heltec — [heltec.org](https://heltec.org/) → продукт **WiFi LoRa 32 (V4)** → раздел документации / Downloads. Прямая ссылка на PDF может быть в описании платы на AliExpress/Amazon или в репозитории Heltec на GitHub.

В документации проекта (например [HELTEC_V4_BOARD_AND_CONFIG.md](../../docs/board/HELTEC_V4_BOARD_AND_CONFIG.md)) пины и константы сверяются с разделами 2.2.1–2.2.3 даташита (J2, J3, Additional Pins).
